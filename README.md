## Quick Start

- Install Nodejs
- Clone the repo
- Install dependancies
- Run initial build
- Start app
     
        npm install
        bower install      
        grunt develop
 
 
##  Personalize your dev environment

Rename the `.env.sample` file to `.env` to change the settings for **MongoDb**, **Mailchimp**, **Mandrill**, **Stripe**, **Hull** and **Meetup**.
